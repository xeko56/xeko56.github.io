[Sudoku Solver](#title)

**Create**: Gib die Ziffern deines Sudokurätsels ein und klick auf "Solve"!
Man kann entweder die Knöpfe klicken oder einfach das Rätselsstring in Inputfeld eingeben, dann kann man sicherlich viel Zeit´sparen.

**Play**: Hast du kein Rätsel zur Hand, kannst du dir eines von den Demo-Sudokus erzeugen lassen.
